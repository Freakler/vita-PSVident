# PSVident

An Application that displays various Information of your Console

## Controls
CROSS - Refresh <br>
SQUARE - Censored Mode <br>
TRIANGLE - Raw Values Mode <br>
CIRCLE -Reset <br>
RIGHT - Next Page <br>
LEFT - Previous Page <br>
DOWN - Save Report ("ux0:data/psvident.txt") <br>
